# Hermes Vault 🧠

Личная база знаний, созданная с помощью Obsidian и Hermes.

## Структура

```
Hermes Vault/
├── 📥 Inbox/           # Входящие заметки (быстрые записи)
├── 📔 Daily/           # Ежедневные заметки (довнические записи)
├── 📝 Notes/           # Основные заметки
├── 📂 Projects/        # Проекты
├── 🏷️ Tags/            # Папка для тегов (опционально)
├── 📎 Attachments/     # Картинки, файлы, PDF
└── 📋 Templates/       # Шаблоны заметок
```

## Быстрый старт

1. **Inbox** — пиши всё что приходит в голову
2. **Daily** — ежедневные записи с датой: `YYYY-MM-DD`
3. **Notes** — постоянные заметки с развитыми мыслями
4. **Projects** — активные проекты с дедлайнами
