# Как пользоваться Obsidian 👋

## Базовый синтаксис

### Заголовки
```markdown
# H1
## H2
### H3
```

### Списки
- Пункт 1
- Пункт 2
  - Подпункт

### Галочки
- [ ] Не сделано
- [x] Сделано

### Ссылки
- Внутри vault: `[[Название заметки]]`
- Внешние: `[текст](https://example.com)`

### Жирный/курсив
- **Жирный**
- *Курсив*
- ~~Зачеркнутый~~

---

## Горячие клавиши

| Действие | Клавиша |
|----------|---------|
| Быстрый поиск | `Ctrl+O` |
| Новая заметка | `Ctrl+N` |
| Командная панель | `Ctrl+P` |
| Ссылка на заметку | `[[` |
| Предпросмотр | `Ctrl+E` |

---

## Полезные плагины

- **Daily Notes** — автоматические дневные записи
- **Graph View** — визуализация связей
- **Templates** — шаблоны заметок
